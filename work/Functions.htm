Functions

  A section of code that performs a specific task

  Examples: main, printf and scanf etc.

  Prototype

        Declaring a function in respect to the scope of the main()

        Not required if function is defined before the main()

  Definition

        The code that makes up the function

  Implementation

        Code that makes use of the function

Example:

//adding two numbers

#include<stdio.h>

int addTwoNumbers(int, int);//function prototype

int main()

{

int x = 5;

int y = 10;

int z = addTwoNumbers(x, y);//function implementation

printf("%i + %i = %i\n",x,y,z);

return 0;

}

int addTwoNumbers(int num1, int num2)//function definition

{

return num1 + num2;

}

 

 

  Name of the function should be same and return type in function and in main should be same

  z will be equal to 15

 

Functions: Passing Variables

  Value

        Creates a copy of the variable to send to the function

When the variable value is changed in the called function the value in the calling function is not modified

  Reference

        Creates a pointer to the variable memory location

When the variable value is changed in the called function the value in the called function is modified

Example on arrays:

//arrays

#include<stdio.h>

{

a[i] = -1;

Static Variable Declaration:

{

{

Arrays

 

  Array is the list of common elements.

        Example : Character Array of size 5

       char [5] creates 5 little boxes.

 

  One Dimensional Array

Declaration : <Variable Type> <Variable Name> [<size>];

char Name [5];/*creates 5 little boxes*/

Name [0] = W; /*Numbering always start from zero*/

Name [1] = i;

And so on.

 

  Two Dimensional Array

 

Declaration : <Variable Type> <Variable Name> [<#Rows>] [<#Columns>];

 

int additionTable [2] [5];

Numbering of rows and columns start from zero.

 

  Multi-Dimensional Arrays

 

Two Dimensional Array

float twoDimArr [5] [10];

 

Three Dimensional Array

long threeDimArr [8] [20] [50];

 

Four Dimensional Array

int fourDimArr [2] [4] [6] [8];

 

  CStrings

  char* mystring = infiniteinsti.com; /*character array is created and mystring is a pointer to infiniteinsti.com*/

  printf(%c, mystring[0]); /*prints infiniteinsti.com*/

 

  Difference between char* and char[] ?

 

        Similarities:

Single Dimensional Array

Pointers

Each character is accessible

        Differences:

char* is static and globally allocated.

char[] is editable and is locally allocated.

Signed & Unsigned Variables:

Specifies type of integer either positive or negative

1. signed - represents integer set

e.g.: signed int x=10;                      // gives 10

    signed int y=-10;                     //gives -10

 

2. unsigned - only +ve including 0

e.g.: unsigned int x=10;                 // gives 10

    unsigned int y=-10;                //gives 10

 

Note: signed int in practice used as int since it specifies entire integer set.

 

Constants:

A variable can be declared to have a constant value throughout the execution by assigning it a parameter.

e.g.:

const int x=10;// where ever you put x it takes 10

Rather than using this method you can also use in the following manner

#define  x =10

This will also do the same work.

We can also escape a tedious work of repetition by using this.

#define   <Variable> <function>

e.g.:

#define correctness printf("This is correct\n");

In the entire program wherever you put the correctness it will print "This is correct"

 

Type Casting:

For example if we do the following

int x=5;

int y=2;

float z=x/y; // give out result 2 which is wrong

 

So to get the appropriate value since the parameters passed are integers and we are expecting a float, we need to make appropriate changes to get that

 

float z=(float)x/(float)y; //this will give out value 2.5

 

Similarly for getting an integer by passing character will be

char ch=X;

int c=(int)ch; //gives out ASCII value of character X

 

How to pass an integer value to Cstring?

Here comes a new variable type char* for string.

Example:

char* deca;

int d=100;

sprintf(deca,%d,d); //this will assign string deca a value 100

printf(deca=%s\n,deca); //this will print 100 on to the console
